# Electric-Vehicle-Data-Analysis
This comprehensive dashboard consolidates electric vehicle data 
spanning multiple dimensions including Battery Electric Vehicles 
(BEVs) and Plug-in Hybrid Electric Vehicles (PHEVs). The analysis 
covers temporal trends across model years, geographic distribution 
across states, and market segmentation by manufacturer and vehicle 
models.
Our data-driven approach provides actionable insights into the 
rapidly evolving EV landscape, enabling stakeholders to understand 
adoption patterns and make informed strategic decisions
